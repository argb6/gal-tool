# 贡献指南

欢迎补充你发现的 Galgame 工具和网站链接！

## 提交方式

### Issue 推荐
在 [Issues](https://github.com/argb6/gal-tool/issues) 中提交，写明：
- 名称
- 链接
- issue标签选择推荐
- 归类（工具 / 网站）

## 格式规范

```markdown
- [名称](链接) 
```

例如：
```markdown
- [VNDB](https://vndb.org/) 
```

## 收录标准

- 与 Galgame 直接或间接相关
- 链接可访问
- 不收录色情、违法内容

## 社区准则

尊重版权，友好交流，为爱发电
