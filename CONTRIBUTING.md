# 贡献指南

欢迎补充你发现的 Galgame 工具和网站链接！

## 提交方式

### Issue 推荐
在 [Issues](https://github.com/argb6/gal-tool/issues) 中提交，写明：
- 名称
- 链接
- 一句话说明
- 归类（工具 / 网站，具体类型）

### Pull Request
1. Fork 本仓库
2. 在对应分类的 `.md` 文件中添加链接条目
3. 提交 PR

## 格式规范

```markdown
- [名称](链接) - 一句话说明
```

例如：
```markdown
- [VNDB](https://vndb.org/) - 英文 Galgame 资料数据库，含评分、角色、会社信息
```

## 收录标准

- 与 Galgame 直接相关
- 链接可正常访问
- 不收录盗版下载站和破解工具
- 不收录色情、违法内容

## 社区准则

尊重版权，友好交流。
