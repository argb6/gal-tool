## 简介

Galgame 玩家向的工具与网站导航仓库。汇总整理了各种实用的 Galgame 相关工具和优质网站链接，方便玩家快速找到所需资源。

只做导航，不做开发。所有收录的内容均为外部已有工具和网站。
