# Gal-Tool - Galgame 工具与网站资源导航

![思维导图](https://github.com/argb6/gal-tool/raw/main/assets/mindmap.png)

## 项目简介

**Gal-Tool** 是一个面向 Galgame 玩家的工具与网站资源导航仓库。整理收录各类实用的 Galgame 相关工具（评分记录、进度管理、图鉴查询、存档下载等）和优质网站资源（资料数据库、社区论坛、攻略站点等），帮助玩家更高效地享受 Galgame。

## 收录内容

### 🛠️ 实用工具

- **评分与记录**：作品评分、游玩进度追踪、已读/未读标记工具
- **图鉴与数据库**：角色图鉴、CG 收藏、会社/声优/画师资料查询
- **存档管理**：存档备份、下载、跨设备同步工具
- **信息查询**：游戏时长查询、攻略路线、分支选项工具

### 🌐 网站资源

- **资料数据库**：作品信息、评价数据、发售日历（如 VNDB、批评空间、Bangumi）
- **社区论坛**：国内外 Galgame 讨论社区、同好交流平台
- **攻略站点**：全路线攻略、选项分支图、隐藏要素解锁指南
- **资讯与评测**：新作资讯、发售情报、中文代理消息、玩家评测

## 项目结构

```
gal-tool/
├── tools/                  # 工具目录
│   ├── rating/            # 评分记录工具
│   ├── progress/          # 进度管理工具
│   ├── archive/           # 存档管理工具
│   └── database/          # 图鉴数据库相关
├── websites/              # 网站资源目录
│   ├── database/          # 资料数据库（VNDB/批评空间等）
│   ├── community/         # 社区论坛
│   ├── guide/             # 攻略站点
│   └── news/              # 资讯测评站
├── assets/                # 资源文件
│   └── mindmap.png        # 项目思维导图
└── docs/                  # 使用指南
```

## 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/argb6/gal-tool.git
   ```

2. **浏览资源**
   - 查看 `tools/` 目录了解各类实用工具
   - 查看 `websites/` 目录获取网站资源导航

3. **参与贡献**
   - 发现好用的工具或网站？欢迎提交 Issue 或 Pull Request
   - 帮忙完善资源信息、修正失效链接、补充使用说明
