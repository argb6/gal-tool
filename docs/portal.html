📖 开发说明

项目结构

  • 仓库： argb6/gal-navigation
  • 数据源： Markdown 文件按类别存放于 simulators/、websites/、tools/ 目录
  • 构建： 纯前端静态页面，无后端依赖
  • 部署： GitHub Pages（自动从 docs/ 目录发布）

技术栈

  • HTML5 / CSS3（暗色主题 #0d1117）
  • 原生 JavaScript（无框架）
  • GitHub REST API（数据拉取）
  • GitHub Actions（链接检查、自动部署）

添加新条目

  1. 在对应类别目录下创建 条目名.md 文件
  2. 按格式填写：
    - [名称](链接) - 描述 `标签1` `标签2`
  3. 提交后约1分钟自动更新到页面

API 调用示例

// 获取所有条目 fetch('https://api.github.com/repos/argb6/gal-navigation/contents/simulators') .then(r => r.json()) .then(data => console.log(data))
