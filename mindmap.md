# Galgame 模拟器导航

```
模拟器
├── krkr
│   ├── 启动方式
│   │   ├── 点击 data.xp3 启动
│   │   ├── 部分游戏有"点我启动.xp3"，可启动中文补丁
│   │   ├── 命名 = data 所在文件夹的名字
│   │   └── 可导入 xp3 和 pfs（artemis）
│   └── bug
│       ├── 连蓝牙耳机熄屏会断连
│       ├── 仅老系统可用，Android 13+ 易闪退
│       └── 苹果不支持
│
├── tyranor
│   ├── 启动方式
│   │   ├── 右上角选择路径，默认在 Tyrano 文件夹
│   │   └── 建议在文件管理主页直接创建
│   ├── 命名 = 游戏文件所在文件夹的名字
│   ├── bug
│   │   ├── 启动界面可添加基础补丁，不行就删了重装
│   │   ├── 仅老系统可用，Android 13+ 易闪退
│   │   └── 苹果不支持
│   ├── 兼容性：比 ONS 好
│   └── 游玩操作：左边游戏 / 右边功能 → 双击 → 菜单 → 按键（右）
│
├── mine
│   ├── 使用方法
│   │   ├── 右下角设置 → 添加游戏路径
│   │   ├── 默认在 Mine 文件夹，建议在文件管理主页直接创建
│   │   └── 更多教程去设置里查看
│   └── bug：蓝牙耳机容易熄屏断连
│
├── winlator
│   ├── 使用方法
│   │   ├── 容积设置
│   │   │   ├── 不知道怎么弄就别乱动
│   │   │   ├── 磁盘管理：第一个默认不动，第二个在 download
│   │   │   ├── 可自行添加磁盘，建议建文件夹放游戏
│   │   │   └── 完成后右下角 √
│   │   ├── 桌面
│   │   │   ├── 对应磁盘找游戏，双击 exe 启动
│   │   │   └── 操作姿势：双击按住鼠标右键
│   │   └── 侧边屏幕
│   │       ├── 可调出输入法
│   │       ├── 任务管理器
│   │       ├── 虚拟按键
│   │       └── 自定义程度很高
│   └── bug
│       ├── 声音有概率消失 → 重启容器 → 换声卡驱动
│       └── 可能无法启动 → 换 GPU 驱动
│
├── 盖世游戏
│   ├── 自由度不如 Winlator，不能改文件
│   ├── 对新手电脑小白友好
│   └── 可玩 Steam，但绑定有点费劲
│
└── 龙神（PC 版 Switch 模拟器）
```

## 说明

本文件为 Galgame 相关模拟器的使用指南与常见问题汇总，收录了在 Android 设备上运行 Galgame 常用的模拟器方案。
